    git log: Эта команда отображает историю коммитов в репозитории. Она показывает список коммитов, начиная с самого последнего, и включает информацию о каждом коммите, такую как хэш коммита, автор, дата и время, а также сообщение коммита. Можно использовать различные флаги, чтобы настроить вывод информации, например, флаг --oneline выводит более краткую информацию о коммитах.

    git show: Эта команда показывает информацию о конкретном коммите или объекте Git. Она показывает изменения, внесенные в коммите, а также информацию о коммите, включая автора, дату и время, и сообщение коммита. Кроме того, git show может также отображать изменения в конкретном файле или папке для данного коммита.

    git blame: Эта команда используется для вывода аннотаций для каждой строки файла, показывая, кто и когда внес изменения в каждую строку. Это полезно для определения, кто именно был ответственным за изменения в конкретном файле. Команда git blame также показывает хэш коммита и сообщение коммита, в котором были внесены изменения.

    git grep: Эта команда позволяет выполнить поиск по содержимому файлов в репозитории, используя регулярные выражения. Она выполняет рекурсивный поиск по всем файлам и выводит строки, соответствующие заданному выражению. git grep может использоваться для поиска определенных строк кода, функций или ключевых слов в проекте.
